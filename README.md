# 📓 Awesome Notebook 

Коллекция из семи самодостаточных реализаций текстового редактора (блокнота) с расширенными возможностями. Каждая версия написана на своём языке программирования и включает:

- ✨ Открытие / сохранение файлов (локальных или через API)
- 🔍 Поиск и замена текста с подсветкой
- 💾 Автосохранение (каждые 30 секунд)
- ⌨️ Горячие клавиши (Ctrl+S, Ctrl+O, Ctrl+F, Ctrl+Z/Y)
- 📊 Статусная строка: кол-во символов, слов, строк
- 🎨 Смена темы (светлая/тёмная) в большинстве реализаций
- 🧩 Drag & Drop файлов (где поддерживается платформой)

## Языки и технологии

| Язык      | Технология / фреймворк        | Тип приложения          |
|-----------|-------------------------------|--------------------------|
| Python    | Tkinter + tkinterdnd2         | Нативное GUI             |
| Rust      | Rocket + Handlebars + jQuery  | Веб-сервер + браузер     |
| Go        | net/http + встроенный шаблон   | Веб-сервер + браузер     |
| Java      | Swing + JEditorPane           | Нативное GUI             |
| C#        | .NET 8 + Windows Forms        | Нативное GUI (Windows)   |
| JavaScript| HTML5 + File API + localStorage| Клиентское веб-приложение|
| PHP       | PHP 8 + нативный файловый ввод | Серверный скрипт (CLI/Web)|
